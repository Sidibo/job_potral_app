# 💼 Job Portal Management System

## 📌 Project Overview
A Job Portal Management System built using Django, Python, Bootstrap, and MySQL/SQLite. It allows users to apply for jobs and admins to manage users, jobs, and applications.

---

## 🚀 Features
- Add/Edit/Delete Users
- Add/Edit/Delete Jobs
- Apply for Jobs
- Prevent Duplicate Applications
- Email Validation
- Responsive Dashboard UI
- Application Tracking

---

## 🛠️ Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- MySQL / SQLite

---

## 🧠 MVT Architecture
- Model → Database
- View → Business Logic
- Template → Frontend UI
